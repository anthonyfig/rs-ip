# rs-ip — Rootstrap AI-delivery method (reusable IP)

Rootstrap's compounding, **non-confidential** intellectual property — the method that travels
across every client engagement. (A client's *Ground Truth content* does not travel; the method does.)

## Contents
- `.claude/agents/` — reusable delivery agents: `ground-truth-guardian`, `capability-spec-writer`, `eval-runner`
- `.claude/skills/ground-truth/` — the skill for reading/writing Ground Truth artifacts
- `explorer/SPEC.md` — the Ground Truth Explorer **app** spec (Ask / Browse / Trace / Validate / Generate / Ingest)
